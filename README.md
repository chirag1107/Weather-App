# 🌤️ WeatherApp

A modern, responsive web application that provides real-time weather forecasts, air quality metrics, and interactive weather visualizations based on user location or search queries.

## 🚀 Features

- **Real-Time Forecasts:** Current temperature, humidity, wind speed, UV index, and atmospheric pressure.
- **Dynamic Visualizations:** Dynamic background changes matching current weather conditions (sunny, rainy, snowy).
- **Geolocation Support:** Automatic weather updates based on the user's current location.
- **7-Day Forecast:** Detailed daily and hourly breakdown of upcoming weather trends.
- **Air Quality Index (AQI):** Tracks PM2.5, PM10, and ozone levels to provide real-time health advisories.
- **Search History & Favorites:** Quickly save and switch between frequently searched cities.
